This document lists worthwile pull requests that haven't been integrated to mainline Flow Graph repository. 

That usually happens for one of following reasons.
* Change is useful to some users but it shouldn't part of the core Flow system. Users might integrate given change to their project code, or turn into another plugin.
* Change might useful to some users but contradicts the Flow Graph design.
